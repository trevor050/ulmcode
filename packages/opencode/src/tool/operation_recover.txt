Restart stale, errored, or cancelled background work for one ULMCode operation.

Use this after `operation_resume`, `runtime_summary`, or `runtime_scheduler` reports stale operation work. The tool scans persisted background metadata for the requested `operationID`, skips currently running jobs, and relaunches restartable model lanes from saved task metadata or supervised commands from saved `command_supervise` metadata.

Set `dryRun` to true to inspect which lanes would restart without launching them.
